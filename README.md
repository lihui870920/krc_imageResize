#A jQuery Plugin to resize images to fit in a container

##Ej:
```
	$(".JSimageBox").pryImageBox({typeOut : true})

```
view in action:	http://goo.gl/9XZOv

##Fatures:
```
Align
Crop
FadeIn anim
All browsers
```

##Options:
```
 typeOut: true,
 animTime: 0,
 responsive: false,
 verticalAling: "CENTER", //"LEFT", "RIGHT"
 horizontalAling: "CENTER" //"TOP", "BOTTOM"
```
